# VSCode 0x96f Material Theme

[0x96f theme for VSCode](https://github.com/filipjanevski/0x96f-vscode-theme) と [Better Material Theme Darker High Contrast](https://github.com/NekoFluff/Better-Material-Theme-Darker-High-Constrast) にインスパイアされたVSCode用ダークカラーテーマです。

## Color Scheme

Better Material Theme Darker High Contrast のカラースキームを参考に、シンタックスハイライトの色割り当てを設計しています。

カラーコード値（パレット）自体は 0x96f 独自のものを維持しつつ、各シンタックス要素への色の割り当てを BMT の配色パターンに合わせています。

### カラーマッピング

| シンタックス要素 | 0x96f の色 | BMT での対応色 |
|---|---|---|
| キーワード/ストレージ | `#A093E2` (紫) | `#C792EA` (紫) |
| 演算子/その他 | `#AEE8F4` (シアン) | `#89DDFF` (シアン) |
| タグ | `#FF7272` (赤) | `#F07178` (赤/ピンク) |
| 関数 | `#49CAE4` (青) | `#82AAFF` (青) |
| 定数/数値 | `#FF8787` (ライトレッド) | `#F78C6C` (オレンジ) |
| 変数パラメータ | `#FF7272` (赤) | `#FF5370` (赤) |
| 文字列 | `#BCDF59` (緑) | `#C3E88D` (緑) |
| クラス/サポート | `#FFCA58` (黄) | `#FFCB6B` (黄) |
| 属性 | `#A093E2` (紫) | `#C792EA` (紫) |
| 正規表現/エスケープ | `#AEE8F4` (シアン) | `#89DDFF` (シアン) |
